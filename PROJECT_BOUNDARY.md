# PROJECT_BOUNDARY.md

## Project identity

This repository is a working tool and site build for Trinity / Faceless Reclaimation-facing material.

It may contain practical interfaces, handouts, visual tools, survival-guide fragments, and implementation experiments connected to the broader ecosystem.

## This project is not

- It is not the canonical Perspection governance repository.
- It is not the public-domain System Refusal Governance Stack.
- It is not allowed to redefine Perspection, Faceless Reclaimation, Trinity, or the Living Survival Guide into one merged identity.

## Allowed material

- Trinity Method / Trinity Framework implementation experiments
- Faceless Reclaimation-facing handouts and explanatory surfaces
- Living Survival Guide tool fragments and charts
- Manus-exported site code that needs sorting
- Practical UI experiments that may later be moved or split

## Not allowed material

- Canonical Perspection governance rules as if this repo owns them
- Perspection entry surface copy that uses guidance, progress, obligation, or recovery framing
- Claims that Perspection is a method, product, healing journey, or instruction sequence
- Copy that collapses separate projects into one identity

## Related but separate

- `7u4n3r/Perspection` is the canonical governance/reference surface.
- `faceless-reclaimation` should hold Reclaimation-facing public materials and lived-experience products.
- `trinity-framework-site` should hold Trinity-specific site material if separated.
- `system-analysis-reference` should hold neutral analysis/reference material.

## Operating rule

Interoperable does not mean merged.

No project may redefine another project's purpose.
